The Josephus problem

Background

The is an example of an algorithm to elect a leader (say, in a system of distributed processors). We use a circular one-directional N node linked-list: each node represents a person who stands in a circle. We start counting at node 1 and always delete the M-th node. Last one standing after (N-1) deletions is the "winner".
